## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Tanya-garg10/portfolio.git
cd portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit:
```
http://localhost:5173
```

## 📦 Build for Production

```bash
npm run build
```

The optimized production build will be in the `dist` folder.

## 🎨 Customization

### Colors
The color scheme uses purple and blue gradients. To customize:
- Edit `src/index.css` for theme colors
- Modify Tailwind config in `tailwind.config.ts`

### Content
- Update personal information in component files
- Add/remove projects in `src/components/ProjectsSection.tsx`
- Modify achievements in `src/pages/Achievements.tsx`
